SOFTVISION - AI-Powered Assistant for the Visually Impaired

SOFTVISION is an Android-based assistive application designed to help visually impaired individuals navigate their surroundings independently. The app uses real-time object detection via the smartphone camera and provides audio feedback to inform users about nearby objects, helping them avoid obstacles and move safely.

- Key Features
1)Real-Time Object Detection - Detects objects using live camera feed and Uses TensorFlow Lite + MobileNet SSD

2) Audio Assistance (Text-to-Speech) - Announces detected objects and Helps users understand surroundings without vision

3) Object Awareness - It provides Object name

4) Fast & Offline Processing - No internet required and Runs efficiently on mobile devices

5) Voice-Based Interaction - User-friendly for visually impaired users

- Tech Stack
1) Mobile Development: Android (Java / Android Studio)
2) Machine Learning: TensorFlow Lite, MobileNet SSD 

- Concepts:
1) Computer Vision
2) Convolutional Neural Networks (CNN)
3) Object Detection

- Results

1) Real-time object detection on mobile
2) Fast inference using TensorFlow Lite
3) Accurate detection using MobileNet SSD
4) Effective audio guidance for navigation

- Challenges
1) Running ML models efficiently on mobile
2) Balancing accuracy vs speed
3) Handling multiple objects in real-time
4) Avoiding repeated audio outputs
   
- Future Improvements
1) GPS-based navigation
2) bstacle avoidance system
3) Smart glasses integration
4) Multi-language voice support
5) Better distance estimation
